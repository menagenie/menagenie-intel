Outlier Lab

Almost there — finish setup to see your dashboard.

This page is the default landing before you've added any creators. Once you run the setup wizard and the refresh, this file is replaced with your live dashboard.

1
In your terminal, from this folder, run:
python3 serve.py
Your browser will open the admin at http://localhost:4747/admin.
2
In the admin: paste your Apify API token, then search for creators and add them. Set your refresh frequency.
3
Click Run refresh now. After ~5–10 min, refresh this tab — you'll see your dashboard with outlier scores, transcripts, and 3-sec hook clips.
4
Commit & push to GitHub. Add APIFY_TOKEN as a repo secret. Deploy to Vercel with vercel. From then on, GitHub Actions runs every Sunday automatically.
Full docs: github.com/ItsssssJack/outlier-lab
